História da Web

Tudo começa em 1944, quando surgiu o primeiro computador, o Mark Ique era basicamente uma máquina de cálculo que funcionava por processos eletromecânicos.

Em 1946, durante a segunda guerra mundial, a Alemanha construiu o Colossus, e os EUA construiram o Eniac para analisar cálculos relacionados à guerra, ambos mais avançados que o Mark I, pois já eram mais automáticos.

Em 1958 surgiu o telégrafo, que foi a primeira forma de comunicação de longas distâncias. Durante a guerra fria, surgiu a necessidade de comunicaçao de forma mais rápida, e entao foi desenvolvida a Arpanet, a primeira conexão dela foi feita em 1969 entre a universidade da Califórnia, e o Instituto de pesquisa de Stanford.

Com o surgimento de várias redes diferentes, houve então a necessidade de criar uma padronização, entao foi criado o WWW, que unia hipertexto e os protocolos TCP/IP.

O que são Clients

Cliente é o dispositivo final quem consome a informação, ou seja, seu computador, seu smartfone, etc. Quem trabalha como FrontEnd, deve se preocupar muito com a Responsividade, que nada mais é do que a compatibilidade da página com os diversos dispositivos e também com os diversos navegadores.

Aplicações web são soluções que usam a internet como meio de comunicação, usando um navegador como intermediário. ou seja, se você entrar no sistema da DIO pelo aplicativo deles, não é uma aplicação web, mas caso você acesse pelo site DIO.me você está usando uma aplicação web.

O que são Servers

Servers (ou servidores), são os computadores centrais de uma rede ou tarefa, geralmente são máquinas bem mais potentes, pois necessitam aguentar o processamento de vários acessos simultâneos. Quando há um grupo maior de computadores atuando como servidor, esses são chamados de Data Centers.

Os servidores podem ser divididos em diferentes tipos:

  • Proxy
  • Firewall
  • DNS
  • Database Server
  • Aplication Server
  • Web Server

Para que um site fique visível para todos, é necessário hospedar o mesmo, que nada mais é do que contratar um servidor da internet, para que ele armazene seus arquivos do site, e os mantenham disponíveis 24hrs por dia para que o site funcione corretamente.

Linguagens de programação

Linguagem de programação é uma linguagem escrita que especifica regras e intruções para gerar softwares e aplicações.

Existem linguagens de programação específicas para rodar tanto do lado do cliente, quanto do lado do Servidor. Alguns exemplos são:

Server Side

  • C#
  • PHP
  • Java
  • JavaScript
  • Ruby

Client Side

  • JavaScript
O fato de que o JavaScript roda dos dois lados, é o que torna essa linguagem uma das mais populares da atualidade.
