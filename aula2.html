Estrutura básica do HTML

A estrutura Básica do HTML consite em 3 tags principais, que são:

< html >...< /html >

É a Tag que identifica o início e o fim do documento, tudo dentro dela será lido e executado.

< head >...< /head >

Tudo dentro dessa tag é executado em segundo plano, ou seja, não é mostrado na tela principal, como parâmetros de linguagem e outras coisas. Ou seja, é a "mente" do nosso código.

< body >...< /body >

É aqui que é escrito toda a estrutura do site e como será mostrado na tela, tudo escrito aqui aparece para o usuário, ou seja, o "corpo" do código.

Aqui eu irei falar sobre as principais Tags que usei para construir esse site.

< h1 > < h2 > < h3 > < h4 > < h5 >

São tags usadas para definir diferentes níveis de título, de acordo com a formatação.

  • < h2>

  • < h3>

  • < h5>

< p >...< /p >

É a tag que serve para escrever textos, exatamente como esse.

< ul >, < ol > e < li >

UL significa Unordered List, ou seja, uma lista em que não mostra a numeração dos elementos, enquanto que OL significa Ordered List, que é uma lista em que é mostrada uma ordem dos elementos. LI é uma tag que aparece dentro das listas, é ela que define cada elemento da lista, não importando se ela é ordenada ou não.

UL
  • Exemplo
  • De
  • Lista
  • Não ordenada
OL
  1. Exemplo
  2. De
  3. Lista
  4. Ordenada

< strong > e < i >

São tags de formatação de texto, que servem para deixar o texto em Negrito (< strong >) e Itálico < I >, também podem ser combinadas dessa forma.

Atributos

Atributos são propriedades que uma Tag pode ter, e são escritos dentro da abertura da tag < tag atributo="valor da propriedade" >. Alguns exemplos de atributos são:

  • Width - Tag que define a largura de um elemento (geralmente de uma imagem).
  • Height - Tag que define a Altura de um elemento (geralmente de uma imagem).
  • Src - Tag usada para definir o caminho de um elemento dentro do código.
  • Href - Define o caminho de um link dentro da Tag < a >.
  • Alt - Usada para descrever uma imagem, para entendimento de leitores de tela ou para caso a imagem não carregue.
  • Style - Tag de CSS, que pode ser usada diretamente dentro do HTML, para estilizar coisas como cores e tamanhos dos elementos da página.

Existem muitas outras Tags e atributos, porém essas foram as principais para a criação desse site.
